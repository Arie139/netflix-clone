## Stack

- Front-end: `HTML, Tailwind CSS and React`
- Back-end: `NodeJS and ExpressJS`
- Database: `Firebase`

## API

- `themoviedb.org`

## Npm Packages

- `Firebase`
- `Tailwind Css`
- `Axios`
- `React-Icons`

## Setup

- Install dependencies with `npm install`.
